# 符号规则

# 光线

  • 从左向右入射方向为正方向

# 球面曲率半径

  • 球心顶点在球面顶点右方为正，反之为负

# 物距

  • 参考点到物点，若沿 Z 时为正，反之为负

# 像距

  • 参考点到像点，若沿 Z 时为正，反之为负

# 物高，像高

  • 均垂直于光轴，向上为正，向下为负

# 角度

  • 以光轴或法线为始边，沿小于 Π 的方向旋转，顺时针为正，逆时针为负

# 折射率

  • 沿 Z 方向传播的光线对应 n 为正，眼 - Z 方向对应 n 为负
