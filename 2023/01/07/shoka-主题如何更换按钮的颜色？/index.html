# shoka 主题如何更换按钮的颜色？

首先膜拜一下 theme shoka 的作者，这个主题真是太棒了！

如题所示，一直没有人出这个教程，今天我解决了，那么我就出这个教程吧。

左侧（或右侧）的主页按钮上的颜色。

通过 F:\blog\themes\shoka\source\css 中的_mixins.styl 更改。

将 135 行中的 orange 和 pink 换个颜色即可，或者改成 16 进制样式。

如（16 进制颜色表达）：

  background-image: linear-gradient(to right,#74ebd5 0,#9face6 100%);

可以和原版对比看：

 background-image: linear-gradient(to right,var(--color-pink) 0,var(--color-orange) 100%);

有的同学可能想把旁边的 more 的颜色也改一下，达到统一。

我想说，当然可以做到。

通过 F:\blog\themes\shoka\source\css_common\components\pages 中的 home.styl 更改。

将 30 行中的 orange 和 pink 换个颜色即可，或者改成 16 进制样式。

如上面所示，再不赘述。

不过一般 41 行和 42 行的：

  box-shadow: 0 0 .6rem .6rem var(--color-pink-a3);
  background-color: var(--color-pink-a3);

改为：

  box-shadow: 0 0 .6rem .6rem var(--color-pink-a1);
  background-color: var(--color-pink-a1);

颜色搭配可能更顺畅一些。当然，自己能调整一下是最好的。

下面就推荐一个比较不错的渐变色网站：免费的渐变背景 CSS3 样式 | oulu.me (pxlab.cn)
