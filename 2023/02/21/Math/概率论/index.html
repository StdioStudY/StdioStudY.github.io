# 连续型随机变量及其概率密度函数

频数

频数是统计学中的一个概念，指的是某个变量值或者某个特征出现的次数。例如，如果一组数据是 {1, 2, 3, 3, 4, 4, 4}，那么数字 4 的频数就是 3，因为它出现了 3 次。

频率

频率是指单位时间内某事件发生的次数 (1)，或者某个组的频数与样本容量的比值 (2)。例如，如果一组数据是 {1, 2, 3, 3, 4, 4, 4}，那么数字 4 的频率就是 3/7，因为它出现了 3 次，而样本容量是 7。

频率密度直方图

频率密度直方图是一种用来表示数据分布情况的图形，它由若干个矩形组成，每个矩形的宽度表示一个组距，高度表示该组的平均频率密度。平均频率密度是指每个单位长度内的频率。频率密度直方图可以反映数据在不同区间内的比例分布情况，也可以看出数据是否符合正态分布或者是否有偏斜。

特点：

  1. 每个小长方形面积等于该组频率
  2. 所有小长方形面积之和等于 1
  3. 介于 x=a 和 x=b 的面积近似等于 [a,b] 之间的频率
